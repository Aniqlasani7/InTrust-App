# Intrust Mobile App

Intrust is a marketplace application that connects customers with verified tradesmen. This app provides a seamless experience for users to find, book, and communicate with tradesmen in various service categories.

## Features

- User authentication (Login, Signup, Onboarding)
- Home screen with search functionality and nearby tradesmen
- Marketplace for browsing tradesmen by categories
- Detailed tradesman profiles with ratings and booking options
- Booking management with confirmation and payment options
- Real-time messaging between customers and tradesmen
- User profile management
- Order history and details

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd intrust-mobile-app
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Run the app:
   ```
   npm start
   ```
